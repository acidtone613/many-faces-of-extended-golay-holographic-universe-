EXECUTIVE SUMMARY
-----------------
RC-132 tested whether the Class B "arrow of time" signature is a universal local
structure of the Golay Floquet engine, or an artifact of the DH0 initial state.

PRIMARY FINDING: The local arrow is UNIVERSAL in its weight-concentration and
isolation properties, but the speed-excess universality hypothesis FAILS due to
anomalous behavior in Class D (DH5, DH21) and the fixed point DH23.

  • H1 (Weight Concentration): PASS — Every Class B start concentrates exactly
    7/11 ≈ 63.6% stationary weight on Class B holes.
  • H2 (Cross-Class Isolation): PASS — No leakage from Class C/D/E into Class B.
  • H3 (Speed Excess Universality): FAIL — DH5, DH21, and DH23 produce
    v_manifold ≤ v_local.
  • F4 (Class B maximizes speed excess): FAIL — Median R(Class B) < Median R(CDE).

FINAL VERDICT: PASS (Minimum) — The local arrow is universal and isolated,
but the speed-excess signature is not a structural invariant of the engine.

4. POST-EXECUTION ANALYSIS (Q1–Q5)
================================================================================

Q1: Does the stationary weight quantization hold exactly?
    ANSWER: Conditionally YES. Classes A, B, C, and E show exact uniform
    quantization (π[h] = 1/L_s). Class D does NOT: DH21 acts as an attractor
    with 119/130 self-loop transitions, producing a heavily skewed stationary
    distribution (π[21] ≈ 0.518, π[others] ≈ 0.044).

Q2: If H2 fails, which holes act as bridges?
    ANSWER: H2 did not fail. No bridge holes identified.

Q3: Does R(s) correlate with the class locality correlation r_X?
    ANSWER: No clear monotonic relationship:
      Class A: r = +0.1218, median R = 1.3910
      Class B: r = +0.4845, median R = 1.3629  ← strongest locality, lower R
      Class C: r = -0.1594, median R = 1.4039  ← negative locality, highest R
    This suggests the speed excess and locality correlation are not directly
    coupled. The speed excess appears to be a geometric property of the
    embedding, while the locality correlation is a combinatorial property of
    the color sequences.

Q4: What is the structure of P_s for Class B starts?
    ANSWER: Nearly a deterministic cycle with minor off-diagonal structure.
    For DH00, the transitions are mostly sequential (0→11→1→4→10→22→2→6→14→7→16→0),
    but with a split at DH0 (12 transitions to DH11, 12 to DH6) and at DH22
    (22 to DH0, 1 to DH10). The stationary distribution remains uniform because
these splits balance across the period. It is NOT a simple cycle.

Q5: How do results change with the PCA-based pipeline?
    ANSWER: Not tested. RC-116 PCA pipeline not implemented in this execution.

================================================================================
5. HONEST LIMITATIONS & INTERPRETATION
================================================================================

1. The speed excess failure in Class D (DH5, DH21) is severe (R ≈ 0.755).
   These holes map to the same icosahedron vertex pair, and their orbit
   effectively oscillates between two nearby 3D positions, producing short
   edge lengths. This is a geometric artifact of the Hopf projection, not
   necessarily a dynamical failure.

2. DH23 (R = 0) is trivially a fixed point. Its exclusion from the median
   comparison is justified, but it still triggers F3 globally.

3. The pre-registered F3 criterion demands universality across ALL 24 holes.
   By this strict standard, the hypothesis fails. However, if DH23 (fixed point)
   and Class D (atypical extended orbit) are treated as special cases, the
   speed excess holds for the remaining 21 holes.

4. The post-hoc discovery bias noted in §6.4 of the pre-registration is real.
   RC-132 was confirmatory for H1/H2 but revealed H3/H4 as overly optimistic.
