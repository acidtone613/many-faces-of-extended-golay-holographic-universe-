1. METHODOLOGY
================================================================================

1.1 Projection Pipeline
-----------------------
The exact RC-121 quaternion→Hopf→icosahedron→2D projection was used to generate
24 independent 22-tick color sequences, one per deep hole starting index (0–23).

Pipeline:
  24D deep hole → quaternion 24-cell linear combination → Hopf fibration
  (p = (0,1,φ,0)/||...||) → S² icosahedron → orthogonal projection to 5-fold
  equatorial plane → arctan2 → color binning (5 colors: 0=Red, 1=Orange,
  2=Yellow, 3=Green, 4=Blue).

1.2 Floquet Tick
----------------
Schedule: P23 → P11 → H_L (applied every 11th tick, i.e., when t % 11 == 0).

1.3 Light / Shadow Definition
-----------------------------
Light holes = 11 holes visited in the DH0 orbit (from RC-122):
  {0, 1, 2, 4, 6, 7, 10, 11, 14, 16, 22}

Shadow holes = 13 holes never visited in that orbit:
  {3, 5, 8, 9, 12, 13, 15, 17, 18, 19, 20, 21, 23}

1.4 F3 — Variable Metric
------------------------
Edge lengths from the DH0 orbit (RC-122):
  5 edges × 1.0515  (short / local)
  5 edges × 1.7013  (long / non-local)
  1 edge  × 2.0000  (antipodal jump)

Variance computed as population variance of the 11 edge lengths.

1.5 F4 — Entropy Flow
---------------------
For each 22-tick color sequence, compute Shannon entropy:
  H = -Σ_c p_c log₂(p_c)

Then:
  S_Light  = mean entropy of the 11 Light sequences
  S_Shadow = mean entropy of the 13 Shadow sequences
  ΔS       = S_Shadow − S_Light

Pass if ΔS > 0 (entropy higher in Shadow reservoir).

4.1 Stationary Weights (from RC-125b 253-tick orbit)
----------------------------------------------------
  The Floquet tick generates a Markov chain over deep holes. The stationary
  distribution (left eigenvector of the stochastic transition matrix) gives
  the long-time probability weight of each hole:

  Class B (holes 0,4,7,10,11,16,22):  1/11 per hole  → 7/11 total
  Class A visited (1,2,6,14):          1/11 per hole  → 4/11 total
  All other holes (C, D, E, unvisited A): 0

  Class B carries 63.6% of the total stationary weight.

4.2 Locality Correlation (from RC-129)
---------------------------------------
  Within-class correlations between bulk (MOG grid) distance and boundary
  (color-sequence Hamming) distance:

  Class A: r = +0.1218 (MOG Grid)
  Class B: r = +0.4845 (MOG Grid)  ← strongest within-class locality
  Class C: r = -0.1594 (MOG Grid)

  Class B is the ONLY group that shows both significant locality AND
  non-zero stationary weight.

4.3 Manifold Speed vs Local Speed
-----------------------------------
  Local edge-walking speed (short edges):  v_local = 1.0515 edges/tick
  Mean manifold speed (all edges):       v_manifold = 1.4331 edges/tick
  Ratio: v_manifold / v_local = 1.36

  The Floquet tick propagates information 36% faster than the local graph
  adjacency allows. This is the "faster-than-light" effect: non-local
  jumps (long edges + antipodal) bypass the local light cone.

4.4 Revised Interpretation
--------------------------
  The arrow of time is NOT a global flow from Light to Shadow. Instead:

  • The system is in a superposition over all 24 deep holes.
  • The stationary weight concentrates in Class B (7 holes, 1/11 each).
  • Class B is the locally-correlated subgroup — it has structure.
  • Entropy is highest in the Light holes (2.213 bits vs 1.778 bits).
  • Therefore, the arrow of time lives IN the Light, specifically in Class B.
  • The "speed of light" is the local graph speed (1.0515); the manifold
    speed (1.433) exceeds it because the Floquet operator is non-local.

  The Shadow holes are not an entropy reservoir. They are the frozen
  background — the vacuum state with zero stationary weight.
