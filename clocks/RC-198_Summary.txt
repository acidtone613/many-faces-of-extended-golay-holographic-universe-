WHAT IS "TRUE TIME" IN THE 4D LAYER?
------------------------------------
  True time is NOT a single clock. It is an EMERGENT PROPERTY of multiple
  clocks with different drift rates, producing entropy and irreversibility.

  The 4D layer has three types of temporal behavior:

  TYPE A: Perfectly Periodic (No Time)
    • Z46 master clock (12D): drift = 0, JS = 0
    • Color entropy (2D): drift = 0, JS = 0
    → These are "clockwork" — they tick but do not evolve.

  TYPE B: Entropy-Producing (True Time)
    • Chiral norm mean/var (6D): ΔS > 0, JS > 0
    • Mass stripped (-9D): ΔS > 0, JS > 0
    → These clocks TICK DIFFERENTLY each cycle. They produce entropy.
    → This is the classical arrow of time.

  TYPE C: Entropy-Consuming (Anti-Time)
    • Dark matter entropy (DM): ΔS < 0, JS > 0
    → This clock RUNS BACKWARD thermodynamically. It absorbs entropy.
    → This is the "ordering engine" that counteracts the arrow of time.

  TYPE D: Slaved Periodic (No Independent Time)
    • QGP entropy (8D): JS > 0 but ΔS = 0
    • Information entropy (4D): JS > 0 but ΔS = 0
    → These are driven by the deep hole orbit (period 22). They show
      irreversibility in their spatial distribution but not in their
      temporal entropy production.

THE ARROW OF TIME IS A SPECTRUM
--------------------------------
  The framework does not have a single arrow of time. It has a SPECTRUM:
    → 12D (Z46): No arrow (perfectly periodic)
    → 8D/4D (QGP/Info): Weak arrow (irreversible distribution, no entropy prod.)
    → 6D (Chiral): Strong arrow (entropy production + irreversibility)
    → -9D (Matter): Strong arrow (entropy production + irreversibility)
    → DM: Negative arrow (entropy consumption)

  The NET arrow of time is the SUM of all these contributions:
    Net ΔS = +0.089 (chiral_mean) + 0.128 (chiral_var) + 0.036 (matter) - 0.169 (DM)
           = +0.084 bits/cycle

  The net arrow is POSITIVE but small. The dark matter sector partially
  cancels the classical arrow of time. This may explain why the observed
  arrow of time in our universe is weak — dark matter acts as a
  thermodynamic "brake."

DISTINCTION FROM QUANTUM TIME (12D)
------------------------------------
  RC-197/RC-197b showed that quantum entanglement (CHSH violation) lives in
  the 48D symplectic space (12D+). RC-198 shows that the ARROW OF TIME lives
  in the 4D–6D layer.

  This is a fundamental separation:
    • Quantum time = entanglement, superposition, non-locality (12D+)
    • Classical time = drift, entropy, irreversibility (4D–6D)

  The 4D layer has "true time" because it is CLASSICALLY COMPLEX but not
  quantum entangled. The complexity produces entropy. The entropy produces
  an arrow. The arrow is irreversible.
