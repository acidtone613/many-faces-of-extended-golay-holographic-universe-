================================================================================
HYPOTHESIS RESULTS
================================================================================

H1 (Pure Engine Orbits):
  PREDICTION: The 5 classes are pure engine orbits.
  RESULT: PARTIAL. The pure engine has 4 orbits (O1, O2, O3, O4), not 5.
  H_L splits O1 into Classes A and B, and O2 into Classes A and C.
  O3 combines with O2 to form Class D. O4 remains Class E.

H2 (Quaternion 24-Cell Invariants):
  PREDICTION: Unit vs half vertices or sign patterns explain classes.
  RESULT: FAIL. All classes mix unit and half vertices. No geometric invariant
  cleanly separates the 5 classes.

H3 (S-Involution Preserves Class):
  PREDICTION: ≥6/12 antipodal pairs share the same class.
  RESULT: FAIL. Only 2/12 pairs share class. The S-involution does NOT
  preserve orbit class membership.

H4 (Color Sequence Statistics):
  PREDICTION: Statistically distinct signatures per class.
  RESULT: PARTIAL. Only Classes D and E have distinctive signatures.
  Classes A, B, C overlap in entropy, dominant color, and color counts.

H5 (P11 Cosets):
  PREDICTION: Half-universes B and C match cosets of <12> in (Z/23Z)*.
  RESULT: FAIL. Both half-universes mix elements from both cosets.

2. H_L INVOLUTION: Applied at ticks t = 0, 11, 22, ...
   Maps between pure engine orbits, creating branching.

CLASS A (8 holes): Start in O1 or O2. Trajectory explores BOTH orbits
  before H_L can trap it. Full mixing → 22 visited holes, period 22.
  Members: {1, 2, 6, 8, 14, 17, 19, 20}

CLASS B (7 holes): Start in O1. Trajectory stays within O1.
  Visited = O1 = 11 holes, periods 3-11.
  Members: {0, 4, 7, 10, 11, 16, 22}

CLASS C (6 holes): Start in O2. Trajectory stays within O2.
  Visited = O2 = 11 holes, periods 3-8.
  Members: {3, 9, 12, 13, 15, 18}

CLASS D (2 holes): Start in O2 (DH5) or O3 (DH21). H_L maps between
  O2 and O3, creating a mixed orbit of 12 holes. Periods 1, 12.
  Members: {5, 21}

CLASS E (1 hole): Start in O4 (DH23). Fixed point of all operations.
  Visited = {23}, period 1.
  Member: {23}
