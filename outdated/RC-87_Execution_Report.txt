4. HYPOTHESIS TESTING
================================================================================

Four hypotheses were tested for deriving the exact tick 1453 = 2×726 + 1

4.1 Hypothesis A: Phase Factor e^(2πi/3)
-----------------------------------------
Hypothesis: The class group action introduces a phase factor e^(2πi/3) in
the perturbation, modifying the splitting.

Test: split_class = split × |1 + e^(2πi/3)| = split × 1.0000000000

Result: The magnitude |1 + e^(2πi/3)| = 1 exactly. No change to perturbation.
Predicted tick: 1452 (unchanged)
Predicted ratio: 1453/1451, error 9.48×10⁻⁷

VERDICT: REJECTED — no modification to perturbation scale.

4.2 Hypothesis B: Three Ideal Classes as Distinct Perturbations
---------------------------------------------------------------
Hypothesis: The three ideal classes correspond to three distinct perturbation
operators V₁, V₂, V₃, and the physical operator is their sum weighted by
class number invariants: V_physical = (V₁ + V₂ + V₃)/3.

Test: Division by 3 modifies ε → ε/3, changing tick from 726 to 726×3/2 = 1089.

Result: Predicted tick = 1089
Predicted ratio = 1.0018382353, error 4.60×10⁻⁴

VERDICT: REJECTED — gives 1089, not 1453.

4.3 Hypothesis C: Stickelberger Relation
----------------------------------------
Hypothesis: The class number 3 enters through the Stickelberger relation
between the class group and Gaussian periods. The Stickelberger element
θ ∈ Z[G] acts on periods to produce a relation where the '3' appears in
the perturbation scale as a multiplicative factor.

Test: If the factor 3 modifies the tick multiplicatively: tick = 3 × (2×726)

Result: Predicted tick = 4356
Predicted ratio = 1.0004592423, error 9.17×10⁻⁴

Analysis: The factor 3 is ALREADY present in the Gram gap (6 = 2×3) and
in 726 = 6×11². No ADDITIONAL factor of 3 appears in the perturbation scale.
The '+1' in 1453 = 2×726 + 1 is not explained by multiplicative factors.

VERDICT: REJECTED — factor 3 is already accounted for in Gram gap.

4.4 Hypothesis D: Regulator / Analytic Class Number Formula
-----------------------------------------------------------
Hypothesis: The exact tick is derived from the regulator or analytic class
number formula: h = (w√|D| / 2πR) × L(1,χ).

For Q(ζ₂₃): h = 3, w = 46, D = 23, R = regulator.
The formula involves 2π and √23, which appear in the Gaussian periods.

Test: The L-value for Q(√−23) is L(1,χ) = 3π / (2√23) ≈ 0.9826010271.
Various combinations tested: (2π/√23)×3 ≈ 3.93, 726×L(1,χ) ≈ 713.37,
1453×L(1,χ) ≈ 1427.72. None yield 1/726 or relate to 1453.

Analysis: 2π and √23 appear in the framework (via Gaussian periods), but
no arithmetic path connects the regulator formula to the '+1' in 1453.

VERDICT: REJECTED — no quantitative bridge from regulator to tick.
