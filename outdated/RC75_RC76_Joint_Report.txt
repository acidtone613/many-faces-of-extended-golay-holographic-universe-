1. FIELD F_2 AND LINEAR ALGEBRA
================================================================================

1.1 The Field F_2 = {0, 1}
    Addition: a ⊕ b = (a XOR b)
    Multiplication: a ⊗ b = (a AND b)

    All matrix operations in this report are performed over F_2.

1.2 Gaussian Elimination over F_2
    Input: Matrix M of size m × n
    Output: Row-reduced echelon form RREF(M)

    Algorithm:
    pivot_row ← 0
    For col = 0 to n-1:
        Find pivot: smallest row ≥ pivot_row with M[row, col] = 1
        If no pivot found: continue
        Swap M[pivot_row] and M[pivot]
        For each row = 0 to m-1, row ≠ pivot_row:
            If M[row, col] = 1:
                M[row] ← M[row] ⊕ M[pivot_row]
        pivot_row ← pivot_row + 1
        If pivot_row = m: break
    Return M

1.3 Matrix Power over F_2
    Input: Square matrix M, exponent n
    Output: M^n over F_2

    Algorithm (binary exponentiation):
    If n = 0: return identity matrix
    result ← identity matrix
    base ← M
    While n > 0:
        If n is odd: result ← result · base (mod 2)
        base ← base · base (mod 2)
        n ← n // 2
    Return result

2. SYMPLECTIC VECTOR SPACE
================================================================================

2.1 Pauli Operators as Vectors
    For n qubits, a Pauli operator 
    P = X^{a_0} Z^{b_0} ⊗ X^{a_1} Z^{b_1} ⊗ ... ⊗ X^{a_{n-1}} Z^{b_{n-1}}
    is represented by the vector (a | b) ∈ F_2^{2n} where:
    a = (a_0, a_1, ..., a_{n-1})
    b = (b_0, b_1, ..., b_{n-1})

2.2 Symplectic Inner Product
    For two operators (a|b) and (a'|b'):
    ⟨(a|b), (a'|b')⟩ = a·b' + b·a' (mod 2)

    Two Pauli operators COMMUTE iff their symplectic inner product is 0.
    They ANTI-COMMUTE iff it is 1.

2.3 Symplectic Group Sp(2n, F_2)
    A 2n × 2n matrix S over F_2 is symplectic iff:
    S^T Ω S = Ω
    where Ω = [[0_n, I_n], [I_n, 0_n]]

    Symplectic matrices represent Clifford gates (unitary operators that 
    map Pauli group to itself).
