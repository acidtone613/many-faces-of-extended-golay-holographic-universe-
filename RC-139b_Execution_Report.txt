EXECUTIVE SUMMARY
================================================================================

RC-139b tested whether orbit-class sub-bundles could support a logical qubit
encoding, as proposed in the RC-139 pre-registration's "PASS (Partial)"
scenario.

The verdict is PASS (Partial) — with a critical refinement.

The original orbit-class sub-bundle approach FAILS because Floquet does not
preserve individual orbit classes. Cross-class transitions occur in all classes
except E.

However, a REFINED approach succeeds: defining sections that are invariant
under the Floquet permutation. Floquet decomposes the 24 holes into 7 cycles
(lengths 3, 4, 4, 4, 4, 4, 1). Sections with constant phase on each cycle
are Floquet-invariant and support a logical qubit encoding.

This is not an orbit-class sub-bundle. It is a "cycle-invariant bundle" over
the full 24-hole base.

H1 — Floquet Permutation Structure
  Result: Analyzed
  Evidence: The Floquet tick induces a permutation on the 24 deep holes with
    cycle decomposition:
    - Cycle 1: [0, 11, 22] (length 3)
    - Cycle 2: [1, 17, 7, 10] (length 4)
    - Cycle 3: [2, 19, 4, 16] (length 4)
    - Cycle 4: [3, 20, 6, 18] (length 4)
    - Cycle 5: [5, 21, 12, 15] (length 4)
    - Cycle 6: [8, 14, 13, 9] (length 4)
    - Cycle 7: [23] (length 1)
  Interpretation: Floquet does not preserve orbit classes. It mixes them
    according to the cycle structure above.

H2 — U_θ Preserves Orbit Class Membership
  Result: PASS
  Evidence: For all 5 orbit classes, U_θ with θ = π/23 preserves class
    membership for both exact deep holes and perturbed states.
  Interpretation: U_θ acts locally and does not mix orbit classes.

H3 — Uniform Phase Rotation Within Each Orbit Class
  Result: PASS
  Evidence: For all 5 orbit classes, U_θ shifts the fiber angle φ by exactly
    θ = π/23 (error < 10^-6) for all deep holes in the class.
  Interpretation: The fiber coordinate is well-defined within each class.

H4 — Logical Qubit Encoding Using Orbit-Class Sub-Bundles
  Result: FAIL
  Evidence: For all viable classes (A: 8 holes, C: 6 holes, D: 2 holes),
    Floquet does not map within the class. The section structure is not
    preserved.
  Interpretation: The orbit-class partition is not the correct sub-bundle
    structure.

H5 — Floquet-Invariant Sections
  Result: PASS
  Evidence:
    - Floquet decomposes the 24 holes into 7 cycles.
    - Sections with constant phase on each cycle are Floquet-invariant.
    - |0_L>: all cycles phase 0
    - |1_L>: alternating cycles phase 0/π
    - Overlap: |<0_L|1_L>| = 0.941 (distinguishable)
    - U_θ fidelity: 1.000000 for both sections
    - Floquet invariance: exact (by construction)
  Interpretation: A logical qubit CAN be encoded using Floquet-cycle-invariant
    sections over the full 24-hole base.

================================================================================
KEY FINDINGS
================================================================================

1. The Floquet tick decomposes the 24 deep holes into 7 cycles, not 5 orbit
   classes. The cycle structure is the true dynamical partition.

2. U_θ preserves orbit classes but Floquet does not. The two operations
   have different symmetry structures.

3. A logical qubit CAN be encoded using sections that are constant on
   Floquet cycles. This is a "cycle-invariant bundle," not an orbit-class
   sub-bundle.

4. The cycle-invariant bundle has 7 base points (one per cycle), not 24.
   The fiber is the phase circle S^1. The total space is 7 × S^1.

5. This is a coarser base than the 24-hole base, but it is dynamically
   preserved by Floquet.

================================================================================
HONEST LIMITATIONS
================================================================================

1. The overlap |<0_L|1_L>| = 0.941 is not orthogonal. The sections are
   distinguishable but not maximally separated.

2. The cycle decomposition was computed for exact deep holes. For general
   states, the Floquet dynamics may have a different cycle structure.

3. The logical qubit test uses a specific section design (alternating cycle
   phases). Other designs may have better or worse properties.

4. The framework uses real vectors, not complex amplitudes. The "phase" is
   a real angle, not a quantum phase.
