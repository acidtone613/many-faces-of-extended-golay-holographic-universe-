  - REASON: Scalar phase functions define INTEGRABLE connections (total differentials)

            closed loop is identically zero.

MATHEMATICAL PROOF OF OBSTRUCTION
================================================================================

Theorem: For any scalar phase function phi: Cocode -> R, the Pancharatnam
geometric phase accumulated along any orbit of P23 on the cocode is zero.

Proof:
  1. P23 acts on cocode cosets as a permutation with cycles of length
     dividing 23 (verified: 2 fixed points + 178 23-cycles).
  2. For a 23-cycle C = (c_0, c_1, ..., c_22), define states
     |Psi_k> = exp(i*phi(c_k)) |chi_{c_k}>, where |chi_c> is the uniform
     state over syndrome subspace c.
  3. The connection gives <Psi_k | Psi_{k+1}> = exp(i*[phi(c_{k+1}) - phi(c_k)]).
  4. The dynamical phase sum = sum_{k=0}^{22} [phi(c_{k+1}) - phi(c_k)]
                            = phi(c_23) - phi(c_0) = 0 (since c_23 = c_0).
  5. The total phase <Psi_0 | Psi_23> = 1 (same state).
  6. Therefore, gamma = arg(1) - 0 = 0.    QED.

Corollary: A non-zero geometric phase requires a NON-INTEGRABLE connection.
This needs either:
  (a) A continuous interpolation U(t) with non-trivial curvature
  (b) A vector bundle with non-Abelian structure group
  (c) A matrix-valued (non-scalar) connection
None of these are naturally present in the discrete P23 action on cocode cosets.
