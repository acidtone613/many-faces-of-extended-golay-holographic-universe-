2.1 THE 24-CELL AND W(F₄)
-------------------------

The 24-cell is the convex hull of the 24 points (±1, ±1, 0, 0) and all
permutations. These are the roots of the D₄ Lie algebra.

Its full symmetry group is the Weyl group W(F₄) of order:

    |W(F₄)| = 1152 = 2⁷ × 3²

The 24-cell has:
    • 24 vertices (8-regular vertex graph)
    • 96 edges
    • 96 triangular faces
    • 24 octahedral cells
    • Euler characteristic: χ = 24 − 96 + 96 − 24 = 0 (consistent with S³)

2.2 LEXICOGRAPHIC TRIANGULATION
--------------------------------

Each octahedral cell is split into 4 tetrahedra by choosing the
lexicographically-first body diagonal (connecting the two vertices with
smallest indices in lexicographic order by coordinates).

2.3 STABILIZER SUBGROUP COMPUTATION
------------------------------------

The hyperoctahedral group B₄ (signed permutation matrices) has order:

    |B₄| = 2⁴ × 4! = 16 × 24 = 384

B₄ is a subgroup of W(F₄) with index [W(F₄) : B₄] = 3.

COMPUTATIONAL METHOD:
    • Construct all 384 signed permutation matrices
    • For each matrix, test whether it preserves the lexicographic diagonal
      choice in all 24 octahedral cells simultaneously
    • A symmetry preserves the triangulation if the image of each chosen
      diagonal is the chosen diagonal in the image cell

RESULT:
    The stabilizer in B₄ has exactly 4 elements:

        Element 0:  I  = diag( 1,  1,  1,  1)     [order 1]
        Element 1:     = diag( 1,  1,  1, −1)     [order 2]
        Element 2:     = diag(−1, −1, −1,  1)     [order 2]
        Element 3: −I  = diag(−1, −1, −1, −1)     [order 2]

    All non-identity elements have order 2.
    The stabilizer is isomorphic to the KLEIN FOUR-GROUP Z₂ × Z₂.

GROUP STRUCTURE VERIFICATION:
    • Element 1 × Element 2 = Element 3  ✓
    • Element 1² = Element 2² = Element 3² = I  ✓
    • Commutative: Element 1 × Element 2 = Element 2 × Element 1  ✓

2.4 INDEX ANALYSIS
-------------------

|W(F₄)| = 1152
|Stab| = 4 (estimated; triality symmetries do not preserve lexicographic order)

Index calculations:
    [B₄ : Stab] = 384 / 4 = 96
    [W(F₄) : Stab] = 1152 / 4 = 288 = 3 × 96

FRAMEWORK SIGNIFICANCE:
    96 is a framework number — it is the EXACT number of edges of the
    24-cell. This is not coincidental: the 96 cosets of the stabilizer in B₄
    correspond to the 96 edge-directions in the broken symmetry.

    288 = 2⁵ × 3² = 32 × 9. While 288 is not itself a framework number,
    it factors as 3 × 96, where 3 = [W(F₄) : B₄] is the triality index.
