HYPOTHESIS B: 144-Tick Period Time-Reversal Symmetry
================================================================================

CLAIM: The full 144-tick Floquet period has time-reversal symmetry: either
S[t] = S[143-t] (direct) or S[t] = 4 - S[143-t] (complement).

TEST: For each deep hole, compute 144-tick sequence and compare first/second
halves under both direct and complement involutions.

                  Pass count (≤14) = 1/24 (DH23 only)

INTERPRETATION: The 144-tick period shows NO time-reversal symmetry. Even the
"best" non-trivial cases (DH22, DH4) have >35% mismatch. The full period is
fundamentally asymmetric — there is no half-period "breathing mode."

FALSIFICATION F_B: FAIL (1/24 < 12/24 threshold)

HYPOTHESIS C: Color Complement Closure
================================================================================

CLAIM: The set of 24 color sequences is closed under color complement C: c → 4-c.
Equivalently, complementing a trajectory yields another valid trajectory.

TEST 1: For each sequence S_i, check if C(S_i) is in the set of 24 sequences.
TEST 2: For each DH i, find DH j with complementary initial color and compare
         trajectories: S_j[t] vs 4 - S_i[t].

RESULTS:
  Closure test:    1/24 sequences have their complement in the set (DH23 only)
  Trajectory test: 0/24 trajectories match their complement-init counterpart
                   Mean Hamming distance = 16.67/22

INTERPRETATION: The color complement is NOT a dynamical symmetry. The color
mapping (arctan2 + rounding) is highly nonlinear and breaks any continuous
symmetry that might exist in the underlying 24D vector space. The 5-color
discretization is not merely a coarse-graining — it is a symmetry-breaking
projection.

FALSIFICATION F_C: FAIL (1/24 < 20/24 threshold)

HONEST LIMITATIONS
================================================================================

1. Hypothesis A tested only phase shifts of 5 and 6. Other phases were not
   tested, but the strong failure (1/24) makes it unlikely that any phase
   shift would restore time-reversal symmetry.

2. Hypothesis B tested only identity and color-complement involutions. More
   complex involutions (e.g., combining color complement with index permutation
   or orbit-class swapping) were not tested.

3. Hypothesis C was weakened by the nonlinearity of color mapping. A "soft"
   test comparing angles before discretization might reveal more structure,
   but the hard test (closure under discrete colors) is the relevant one for
the framework's observable dynamics.

4. The thresholds were pre-registered but arbitrary. They represent "strong
evidence" rather than "proof." The actual results (1/24 passing) are so far
from threshold that this limitation is moot.
