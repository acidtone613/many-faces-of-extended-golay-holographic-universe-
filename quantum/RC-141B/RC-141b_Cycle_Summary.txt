1. EXECUTIVE SUMMARY
================================================================================

This document summarizes the two verified cycle decompositions of the Floquet
dynamics on the 24 deep holes of the [[24,12,8]] quantum Golay code:

  • Cycle Set 1 (RC-139b):  Single-tick Floquet operator F  →  7 cycles
  • Cycle Set 2 (RC-141b):  11-tick stroboscopic block B    → 13 cycles

Both decompositions are EXACT permutations (distance = 0.0000000000 for all
24 deep holes). The 13-cycle block B is the square of the 7-cycle operator F
in a specific sense: each B-cycle is a union of F-cycles, and B = F^11.

The 7-cycle base (RC-139b) supports Floquet-invariant sections with exact
invariance, overlap 0.941, and U_θ fidelity 1.000000.

The 13-cycle base (RC-141b) is also a valid dynamical invariant, but the
(0,23) phase fiber is incompatible with B because B swaps component 0 with
component 16.

5. KEY FINDINGS FROM RC-139b AND RC-141b
================================================================================

RC-139b (7-Cycle Base):
  • H1 (Floquet permutation structure): 7 cycles verified
  • H2 (U_θ preserves cycle membership): PASS
  • H3 (Uniform phase rotation): PASS
  • H4 (Orbit-class sub-bundle qubit): FAIL (Floquet mixes orbit classes)
  • H5 (Floquet-invariant sections): PASS
    - Overlap |<0_L|1_L>|: 0.941
    - U_θ fidelity: 1.000000
    - Floquet invariance: EXACT (allclose = True)

RC-141b (13-Cycle Base):
  • H1 (U_θ preserves cycle membership): PASS
  • H2 (B preserves cycle membership): PASS
  • H3 (Uniform phase rotation): PASS
  • H4/H5 (Logical qubit on 13-cycle base): FAIL
    - Overlap |<0_L|1_L>|: 0.941852  (PASS)
    - U_θ fidelity: 1.000000  (PASS)
    - Floquet fidelity |0_L>: 0.989721  (FAIL, threshold 0.99)
    - Floquet fidelity |1_L>: 0.939262  (FAIL)

Root cause of RC-141b failure:
  B swaps component 0 ↔ component 16 for all vectors. The (0,23) phase fiber
  is scrambled because the rotated value in component 0 is moved to component 16.
  Cycle means WITHOUT phase rotation are exactly invariant (diff = 0.0000000000).
  The failure appears ONLY when phase is encoded in component 0.

6. THE HONEST BOTTOM LINE
================================================================================

| Question                        | Answer                                    |
|:--------------------------------|:------------------------------------------|
| What is the correct base?       | The 7 cycles of the single-tick operator F |
| Does the 13-cycle base work?    | It is a valid invariant, but the (0,23)   |
|                                 | phase fiber is incompatible with it.        |
| What works for a logical qubit? | Floquet-cycle-invariant sections over     |
|                                 | the 7 F-cycles (RC-139b H5).              |
| What is the next step?          | Find a phase fiber that is preserved by   |
|                                 | BOTH F and B, or accept the 7-cycle base. |

The 7-cycle base (RC-139b) is the only verified structure that supports a
functional logical qubit encoding in the 24D-DMF framework. The 13-cycle base
(RC-141b) is a coarser, valid dynamical invariant, but it requires a different
phase fiber (e.g., one that treats components 0 and 16 symmetrically).
